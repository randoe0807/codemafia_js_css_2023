### inline要素, block要素の違い

- inlineはwidth, heightを指定できない。styleで指定しても無視される（ただし、styleでdisplay: blockを指定するとblock要素に変更できる）
- blockはwidth,heightを指定できる
- inlineとblockでレイアウトの指定方法が変わることを頭に入れておく
- タグによってinlineかblockかを判断するにタグで囲ったものが、縦に並ぶか、横に並ぶかで見極める
